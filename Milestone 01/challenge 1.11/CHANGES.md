# Changelog - Dev Confessions API

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-09

### Added
- Initial project setup with Express.js
- MVC architecture implementation
- Confession controller with CRUD operations
- Confession routes configuration
- Confession service layer
- Environment configuration support
- Error handling middleware
- Health check endpoint

### Features
- Create new confession
- Retrieve all confessions
- Get confession by ID
- Filter confessions by category  
- Delete confession by ID

### Technical Stack
- Node.js
- Express.js 4.18.2
- REST API architecture
- MVC pattern
